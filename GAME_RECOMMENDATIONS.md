## 🟢 쉬운 게임 (진입 장벽 낮음)

### 1. 오목 (Gomoku) ⭐ 강력 추천

**장점:**
- 룰 초단순 (5개 연속)
- 전세계적으로 인기
- 빠른 게임 (5~10분)
- 개발 쉬움

**구현 난이도:** ⭐⭐☆☆☆  
**개발 시간:** 1주  
**3D 필요:** ❌ (2D 그리드로 충분)  
**물리 엔진:** ❌

**기술 스펙:**
```python
# 게임 상태
state = {
  'board': [[None] * 15 for _ in range(15)],  # 15x15 보드
  'currentTurn': 'black',
  'winner': None
}

# 승리 조건: 5개 연속
def check_win(board, x, y, color):
    directions = [(1,0), (0,1), (1,1), (1,-1)]
    for dx, dy in directions:
        count = 1
        # 양방향 체크
        count += count_direction(board, x, y, dx, dy, color)
        count += count_direction(board, x, y, -dx, -dy, color)
        if count >= 5:
            return True
    return False
```

**저작권:** ✅ 안전 (전통 게임, 퍼블릭 도메인)

## 🟡 중간 난이도 (전략 필요)

### 4. 루미큐브 (Rummikub) ⭐ 강력 추천

**왜 추천?**
- 숫자 조합 게임 (마작 + 포커)
- 전략적이면서도 운 요소
- 한국에서 인기 많음 (럭키투게더 유사)
- 2~4인, 20~40분

**구현 난이도:** ⭐⭐⭐⭐☆  
**개발 시간:** 3-4주  
**3D 필요:** ⭕ (타일이 렉시오랑 비슷)  
**물리 엔진:** 선택 (미끄러지는 연출)

**게임 규칙:**
```
타일: 1-13 숫자 x 4색 + 조커 2개
조합:
  - 같은 숫자 3개 이상 (색 다름)
  - 연속 숫자 3개 이상 (같은 색)

턴 구조:
  1. 타일 배치 (기존 조합 재배치 가능)
  2. 타일 뽑기 (배치 못했을 때)
  
승리: 모든 타일 먼저 내놓기
```

**구현 포인트:**
```typescript
// 타일 조합 검증
function validateSet(tiles: Tile[]): boolean {
  if (tiles.length < 3) return false;
  
  // 같은 숫자?
  const allSameNumber = tiles.every(t => t.number === tiles[0].number);
  if (allSameNumber) {
    // 모두 다른 색?
    const colors = new Set(tiles.map(t => t.color));
    return colors.size === tiles.length;
  }
  
  // 연속 숫자?
  const allSameColor = tiles.every(t => t.color === tiles[0].color);
  if (allSameColor) {
    tiles.sort((a, b) => a.number - b.number);
    for (let i = 1; i < tiles.length; i++) {
      if (tiles[i].number !== tiles[i-1].number + 1) return false;
    }
    return true;
  }
  
  return false;
}
```

**저작권:** ⚠️ "Rummikub"는 상표  
**안전한 이름:** "타일 러미", "숫자 조합 게임"

## 📊 최종 추천 순서

### 즉시 개발 권장 (Phase 1)
```
1. 오목 ✅
   - 개발: 1주
   - 이유: 빠른 개발, 즉시 효과, 전세계 인기
   
2. 루미큐브 (→ 타일 러미) ✅
   - 개발: 3-4주
   - 이유: 전략성, 한국 인기, 차별화
```

### 다음 단계 (Phase 2)
```
3. 할리갈리 (→ 종치기) ✅
   - 개발: 2주
   - 이유: 파티 게임, 캐주얼 타겟
   
4. 체스 ✅
   - 개발: 4주
   - 이유: 글로벌 표준, AI 연습 모드
```

### 나중에 고려 (Phase 3)
```
5. 스플렌더 (→ 보석 수집)
6. 카탄 (완전 재해석 버전)
7. 텔레스트레이션
```

## ✅ 게임 추가 시 절차

   - 룰 정리

   - 저작권 확인

   - `games/[game]/rules.py` 구현
   - 게임 로직 테스트

3. **프론트엔드**
   - `games/[game]/[Game]Game.ts` 구현

4. **테스트**
   - 단위 테스트
   - 통합 테스트
