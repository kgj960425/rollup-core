# 🎮 추가 게임 추천 가이드

## 📋 개요

Rollup 플랫폼에 추가하기 좋은 보드게임 목록을 난이도, 인기도, 개발 공수를 고려하여 정리했습니다.

---

## 🟢 쉬운 게임 (진입 장벽 낮음)

### 1. 오목 (Gomoku) ⭐⭐⭐⭐⭐

**추천 이유**
- 룰 초단순 (5개 연속 놓기)
- 전세계적으로 인기
- 빠른 게임 (5-10분)
- 2인 전용

**구현 난이도**: ⭐⭐☆☆☆

**기술 요구사항**
- 2D 그리드 (19x19)
- 3D 필요 없음 (선택)
- 물리 엔진 불필요
- 승리 조건 체크 (5개 연속)

**개발 시간**: 1주

**백엔드 구현**
```python
class GomokuRules(BaseGameRules):
    def get_config(self):
        return GameConfig(
            id='gomoku',
            name='오목',
            min_players=2,
            max_players=2,
            turn_time_limit=30,
            has_physics=False,
            has_3d_board=False,
            category='board'
        )
    
    def check_win_condition(self, state):
        # 가로/세로/대각선 5개 체크
        pass
```

**프론트엔드 구현**
- 19x19 그리드 렌더링
- 클릭으로 돌 놓기
- 승리 라인 표시

### 2. 커넥트 포 (Connect Four) ⭐⭐⭐⭐

**추천 이유**
- 단순한 규칙
- 중력 메커니즘 (재미 요소)
- 빠른 게임
- AI 구현 쉬움

**구현 난이도**: ⭐⭐☆☆☆

**기술 요구사항**
- 7x6 그리드
- 중력 시뮬레이션 (위에서 떨어짐)
- 3D 애니메이션 (선택)

**개발 시간**: 1주

---

### 3. 할리갈리 (Halli Galli) ⭐⭐⭐⭐

**추천 이유**
- 반응속도 게임 (흥미진진)
- 파티 게임 느낌
- 애니메이션 화려하게 가능
- 2-6인

**구현 난이도**: ⭐⭐⭐☆☆

**⚠️ 주의사항**
- "Halli Galli" 이름 사용 불가 (상표권)
- 대체 이름: "종치기 게임", "순발력 게임"
- 과일 그래픽 자체 제작 필수

**게임 메커니즘**
- 카드 뒤집기
- 같은 종류 5개 → 종 치기
- 가장 빠른 사람이 카드 획득

**개발 시간**: 1-2주

---

### 4. 도미노 (Dominoes) ⭐⭐⭐

**추천 이유**
- 전통 게임 (퍼블릭 도메인)
- 간단한 규칙
- 다양한 변형 가능

**구현 난이도**: ⭐⭐☆☆☆

**개발 시간**: 1주

## 🟡 중간 난이도 (전략 필요)

### 5. 루미큐브 (Rummikub) ⭐⭐⭐⭐⭐

**추천 이유** (최우선 추천!)
- 한국에서 매우 인기 (럭키투게더)
- 숫자 조합 게임 (마작 + 포커)
- 전략적이면서도 운 요소
- 2-4인, 20-40분

**구현 난이도**: ⭐⭐⭐⭐☆

**⚠️ 주의사항**
- "Rummikub" 이름 사용 불가
- 대체 이름: "타일 러미", "숫자 조합 게임"
- 타일 디자인 자체 제작

**게임 메커니즘**
- 타일 조합 (같은 숫자 3개 or 연속 숫자)
- 기존 조합 재배치 가능
- 30점 이상 조합으로 시작

**기술 요구사항**
- 타일 드래그 앤 드롭
- 조합 검증 로직
- 3D 타일 (렉시오 시스템 재활용)
- 물리 엔진 (선택)

**개발 시간**: 2-3주

**백엔드 구현 팁**
```python
def validate_combination(tiles):
    # 타입 1: 같은 숫자, 다른 색 (최소 3개)
    if is_same_number_set(tiles):
        return True
    # 타입 2: 연속 숫자, 같은 색 (최소 3개)
    if is_run(tiles):
        return True
    return False
```

## 🔴 어려운 게임 (복잡한 룰)

### 9. 체스 (Chess) ⭐⭐⭐⭐

**추천 이유**
- 클래식 명작
- 전세계 유저층
- AI 연습 모드 추가 가능
- 퍼블릭 도메인

**구현 난이도**: ⭐⭐⭐⭐☆

**단점**
- 이미 많은 체스 앱 존재 (경쟁 심함)
- 룰 엔진 복잡 (체크메이트, 캐슬링, 앙파상)

**개발 시간**: 2-3주

**장점**
- 라이브러리 활용 가능 (chess.js)
- AI 구현 풍부 (Stockfish)

---

### 10. 카탄 (Catan) ⭐⭐⭐⭐

**추천 이유**
- 보드게임 클래식
- 자원 관리 + 교역
- 리플레이 가치 높음

**구현 난이도**: ⭐⭐⭐⭐⭐

**⚠️ 주의사항 (높은 리스크)**
- "Catan" 상표권 강력하게 보호됨
- 육각형 타일 + 자원 거래 조합 위험
- 대체 이름: "정착지 건설", "개척자들"
- 보드 디자인 완전히 다르게 (사각형 타일 등)

**개발 시간**: 4-5주

**권장**: 법률 자문 후 진행

### 11. 바둑 (Go) ⭐⭐⭐

## 🎯 추천 개발 순서

### Phase 1: 간단한 게임 (검증)
1. **오목** - 가장 빠르게 완성 가능, 유저 반응 확인
2. **할리갈리** (종치기 게임) - 파티 게임, 재미 요소

### Phase 2: 전략 게임 (깊이)
3. **루미큐브** (타일 러미) - 한국 시장 공략, 전략 깊이
4. **체스** - 글로벌 유저층

### Phase 3: 파티 게임 (다양성)
5. **텔레스트레이션** - 소셜 요소 강화
6. **코드네임** (암호명) - 팀 게임

### Phase 4: 복잡한 게임
7. **스플렌더** (보석 수집가)
8. **뱅!** (서부 결투)

---

## 💡 게임 선정 기준

### 높은 점수를 받는 게임
✅ 간단한 규칙 (5분 내 설명 가능)
✅ 빠른 게임 (30분 이내)
✅ 리플레이 가치 높음
✅ 다양한 인원 (2-6인)
✅ 법적 리스크 낮음 (퍼블릭 도메인 or 이름변경)
✅ 차별화 가능 (3D, 애니메이션)

### 낮은 점수를 받는 게임
❌ 복잡한 규칙
❌ 긴 플레이 타임 (1시간+)
❌ 강한 상표권 보호
❌ 기존 온라인 버전 많음 (경쟁)
❌ 2인 전용 (확장성 낮음)
